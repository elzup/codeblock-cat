# md-cat [![Build Status](https://travis-ci.com/elzup/md-cat.svg?branch=master)](https://travis-ci.com/elzup/md-cat)

> cat file with markdown code block format.

## Install

```
$ npm install md-cat
```

## CLI

```
$ npm install --global md-cat
$ md-cat [filename]
```

Examples

```
$ md-cat list.txt
\`\`\`txt titlelist.txt
apple
banana
\`\`\`
```
